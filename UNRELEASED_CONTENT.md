# Unreleased RSC Content Documentation

## Overview
The RSC cache contains numerous items, sprites, and sounds that were **never released** to the live game. These represent scrapped features, developer tools, and planned content that was abandoned.

## Confirmed Unreleased Items

### Necromancy-Related Content
**Life Runes** - Found in game config files
- Intended for summoning spells
- Spell names: "Call Animal", "Raise Skeleton", "Summon Demon"
- Never implemented in RSC
- Eventually became Summoning skill in RS2

### Zamorakian Items
**Bloody Axe of Zamorak**
- Quest item (never used)
- Possibly related to "Hzil Cult" quest
- Exists in cache but no functionality

**Mace of Zamorak**
- Found in data files
- Unknown purpose
- Never released

### Mystery Items
**Special Curry** - Purpose unknown
**Steel Wire** - Purpose unknown  
**Super Chisel** - Purpose unknown
**Soil** - Believed to be predecessor to Compost (RS2 Farming)
**Present** - Purpose unknown
**Info Document** - Possibly developer tool
**Dig Site Info** - Possibly quest-related notes
**Null Amulet** - Added Oct 2005, references "Lord Null" (unreleased quest)

### Decorative/Event Items
**Spinning Plates** - April Fool's joke (2005)
- Dragon picture to trick players
- Some broken versions still exist

**Fireworks** - Scrapped decorative item
**Cooking Pot** - Purpose unknown
**Heraldic Platebodies** - Planned for 2006 Treasure Trails (never released)
**Sleeping Bag** - Found in RS2 beta files
**Void Staff** - Unreleased weapon model
